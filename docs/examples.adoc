[#examples]
= Configuration examples
:imagesdir: media/

Below are some examples of how you can configure the application.

== Single Tenant

Example configuration using Single Directory (Tenant)

.com.enonic.app.entraidprovider.cfg:
[source,properties]
----
autoinit=true
idprovider.entra.oidcWellKnownEndpoint=https://login.microsoftonline.com/12345678-1234-1234-1234-123456789abc/v2.0/.well-known/openid-configuration
idprovider.entra.clientId=********
idprovider.entra.clientSecret=********
idprovider.entra.mappings.displayName=@@{userinfo.given_name} @@{userinfo.family_name}
----

== Multi Tenant

Example configuration using Single Directory (Tenant)

.com.enonic.app.entraidprovider.cfg:
[source,properties]
----
autoinit=true
idprovider.entra.oidcWellKnownEndpoint=https://login.microsoftonline.com/organizations/v2.0/.well-known/openid-configuration
idprovider.entra.clientId=********
idprovider.entra.clientSecret=********
idprovider.entra.mappings.displayName=@@{userinfo.given_name} @@{userinfo.family_name}
idprovider.entra.rules.allowedTenants= 12345678-1234-1234-1234-123456789abc 87654321-4321-4321-4321-cba987654321
----
